効率的でおなじみの文法

ECMAScript5を基本的に従う文法、複数の付加機能を加えました。 四糸乃エンジンでは、オブジェクト指向の要素をさらに強化し、データ型宣言を提供しています。 


"use yoshinon";

class Character implements YoshinonObject {
	public function __constructor(chName, optObject){
		chName = typeof chName !== "undefined" ? chName : "匿名";
		if(typeof optObject !== "object") throw new Error("Type of optObject must be \"object\".");
	}
}

CLIベースの自動管理システム

GitやSVNを使用して、ストーリーを管理することができます。もちろん、バージョン管理ソリューションの利点であるロールバック機能と履歴にも対応しています。 


C:\Users\User> cd Yoshinon
C:\Users\User\Yoshinon> yosgit story add :237 
Story line :237 added successfully!

C:\Users\User\Yoshinon> yosgit story checkout -b dxStory01
Story branch "dxStory01" created successfully!
Branch activation changed to "dxStory01".

分かりやすい変数と関数

キャラクターを生成するためにコアを確認する必要がありません。ただCharacterオブジェクトを生成するだけでノーベルで使用キャラクターを定義することができます。 


var chExample = new Character("例人", {
	"color": "#FF00FF",
	"decoration": "underline"
});

chExample.say("愛しています、皆さん！");
chExample.selection([
	{ "title": "私もそうです！", "description": "私はあなたを愛しています。", "on": function(){} },
	{ "title": "バカじゃないの？", "description": "吉野の精神が飛んだようです。", "on": function(){} }
]);
